# Как создать виртуальное окружение и запустить проект
Все команды выполняются в терминале (командной строке), находясь в папке вашего проекта.

### 1. Создание виртуального окружения
Оно необходимо, чтобы изолировать зависимости этого проекта от вашей остальной системы.

**Windows:**
```bash
python -m venv venv
```

**macOS / Linux:**
```bash
python3 -m venv venv
```

(Здесь venv — это название папки, в которой будет храниться окружение).

### 2. Активация окружения

**Windows (Командная строка - cmd):**
```bash
venv\Scripts\activate.bat
```

**Windows (PowerShell):**
```bash
venv\Scripts\Activate.ps1
```

**macOS / Linux:**
```bash
source venv/bin/activate
```

После успешной активации в терминале слева появится приписка (venv).

### 3. Установка зависимостей

Установите все библиотеки из вашего файла:
```bash
pip install -r requirements.txt
```
