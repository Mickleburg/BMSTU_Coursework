# Лабораторная работа № 8

## Тема

Использование шейдеров OpenGL.

## Основа работы

В качестве основы взята [лабораторная работа №6](../lab6). Сцена и управление сохранены, но расчет освещения и применение текстуры перенесены в шейдеры OpenGL.

В программе используются:

- вершинный шейдер для передачи положения вершины, нормали и текстурных координат;
- фрагментный шейдер для расчета освещения по нескольким источникам;
- uniform-переменные для параметров источников света, материала и состояния текстуры;
- процедурная шахматная текстура из ЛР6.

## Вариант

Переписана на шейдерах лабораторная работа №6.

Из ЛР6 сохранены:

- модель икосаэдра;
- несколько источников света с возможностью включения и выключения;
- движение объекта с абсолютно упругим отражением от границ ограничивающего объема;
- текстура для изменения свойств поверхности;
- переключение каркасного и твердотельного отображения.

## Управление

| Клавиша | Действие |
|---|---|
| `←` / `→` | поворот сцены вокруг оси `Oy` |
| `↑` / `↓` | поворот сцены вокруг оси `Ox` |
| `W` / `S` | приблизить / отдалить камеру |
| `+` / `-` | увеличить / уменьшить масштаб основного объекта |
| `F` | переключить каркасное и твердотельное отображение |
| `T` | включить / выключить текстуру |
| `Space` | пауза / продолжение анимации |
| `R` | сбросить положение, скорость и вращение объекта |
| `1` | включить / выключить первый источник света |
| `2` | включить / выключить второй источник света |
| `3` | включить / выключить третий источник света |
| `B` | показать / скрыть ограничивающий объем |
| `Esc` | выход из программы |

## Структура программы

Основные функции:

| Функция | Назначение |
|---|---|
| `get_icosahedron_geometry` | формирование вершин и граней икосаэдра |
| `face_normal` | вычисление нормали треугольной грани |
| `set_perspective_projection` | ручное задание матрицы перспективной проекции |
| `init_gl` | настройка OpenGL и компиляция шейдерной программы |
| `create_checker_texture` | процедурная генерация текстуры |
| `set_shader_lights` | передача параметров источников света в шейдер |
| `set_shader_material` | передача параметров материала в шейдер |
| `draw_icosahedron` | отрисовка икосаэдра с использованием шейдеров |
| `draw_bounding_box` | отрисовка ограничивающего объема |
| `update_animation` | обновление положения объекта и обработка отражений |
| `key_callback` | обработка клавиатурного управления |
