# Лабораторная работа № 4

## Тема

Алгоритмы растровой развертки многоугольника и фильтрации изображения.

## Вариант

- **Алгоритм растровой развертки:** A5 — заполнение многоугольника со списком рёбер и флагом
- **Алгоритм фильтрации:** Б2 — постфильтрация с взвешенным усреднением области \(N \times N\)
- **Параметр:** \(N = 3\)

## Задание

### 1. Основной алгоритм развертки

Реализовать алгоритм растровой развертки многоугольника в соответствии с вариантом:

- **A5** — заполнение многоугольника со списком рёбер и флагом.

### 2. Алгоритм фильтрации

Реализовать алгоритм фильтрации в соответствии с вариантом:

- **Б2** — постфильтрация с взвешенным усреднением области \(3 \times 3\).

### 3. Вспомогательные алгоритмы

Реализовать необходимые вспомогательные алгоритмы растеризации отрезка с модификациями, обеспечивающими корректную работу основного алгоритма.

### 4. Ввод исходных данных

Предусмотреть интерактивный ввод исходных данных для каждого алгоритма с помощью клавиатуры и/или мыши.

Также должна быть реализована возможность:

- очистки области вывода;
- отмены ввода.

### 5. Растеризация

Растеризацию выполнять в специально выделенном буфере в памяти с последующим копированием результата в буфер кадра OpenGL.

Необходимо предусмотреть возможность изменения размеров окна.

## Что должно быть реализовано

В рамках лабораторной работы необходимо реализовать:

- заполнение многоугольника со списком рёбер и флагом;
- постфильтрацию с взвешенным усреднением области \(3 \times 3\);
- вспомогательные алгоритмы растеризации отрезков;
- интерактивный ввод данных;
- очистку области вывода;
- работу через промежуточный буфер с выводом в OpenGL;
- корректную обработку изменения размеров окна.
