# Лабораторная работа № 6

## Тема

Построение реалистичных изображений трёхмерной сцены средствами OpenGL.

## Основа работы

Базой для лабораторной работы №6 является лабораторная работа №3.

Из ЛР3 сохранены:

- параметризованная модель объекта сцены — икосаэдр;
- трёхточечная перспективная проекция;
- дополнительный икосаэдр в стандартной ориентации для демонстрации проекции;
- навигация камеры с помощью модельно-видовых преобразований без `gluLookAt`;
- переключение между каркасным и твердотельным отображением через `glPolygonMode`.

## Вариант

| Вариант | Реализация |
|---:|---|
| А6 | Использование более одного источника света |
| Б2 | Движение тела с абсолютно упругим отражением от границ ограничивающего объёма |
| В2 | Использование текстуры для определения свойств поверхности |

## Задание

Необходимо расширить сцену из лабораторной работы №3 и добавить средства повышения реалистичности изображения:

1. Задать параметры модели освещения OpenGL:
   - свойства источников света;
   - свойства материала поверхности;
   - параметры глобальной модели освещения.

2. Реализовать вариант **А6**:
   - определить несколько источников света;
   - задать для каждого источника положение и компоненты `ambient`, `diffuse`, `specular`;
   - предусмотреть возможность включения и выключения отдельных источников.

3. Реализовать вариант **Б2**:
   - задать ограничивающий объём регулярной формы;
   - реализовать движение икосаэдра с начальной скоростью;
   - обеспечить абсолютно упругое отражение объекта от границ объёма.

4. Реализовать вариант **В2**:
   - создать или загрузить текстуру;
   - использовать текстуру для изменения свойств поверхности;
   - предусмотреть возможность отключения текстурирования.

## Управление

| Клавиша | Действие |
|---|---|
| `←` / `→` | поворот сцены вокруг оси `Oy` |
| `↑` / `↓` | поворот сцены вокруг оси `Ox` |
| `W` / `S` | приблизить / отдалить камеру |
| `+` / `-` | увеличить / уменьшить масштаб основного объекта |
| `F` | переключить каркасное и твердотельное отображение |
| `T` | включить / выключить текстуру |
| `Space` | пауза / продолжение анимации |
| `R` | сбросить положение, скорость и вращение объекта |
| `1` | включить / выключить первый источник света |
| `2` | включить / выключить второй источник света |
| `3` | включить / выключить третий источник света |
| `B` | показать / скрыть ограничивающий объём |
| `Esc` | выход из программы |

## Структура программы

Основные функции:

| Функция | Назначение |
|---|---|
| `get_icosahedron_geometry` | формирование вершин и граней икосаэдра |
| `face_normal` | вычисление нормали треугольной грани |
| `set_perspective_projection` | ручное задание матрицы перспективной проекции |
| `init_gl` | начальная настройка OpenGL |
| `create_checker_texture` | процедурная генерация текстуры |
| `apply_lights` | настройка трёх источников света |
| `set_surface_material` | задание свойств материала |
| `draw_icosahedron` | отрисовка икосаэдра |
| `draw_bounding_box` | отрисовка ограничивающего объёма |
| `update_animation` | обновление положения объекта и обработка отражений |
| `key_callback` | обработка клавиатурного управления |
