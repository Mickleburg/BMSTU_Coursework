# Репозиторий учебных курсов МГТУ им. Баумана (BMSTU_Coursework)

Этот репозиторий создан для хранения и систематизации лабораторных работ, проектов и других материалов, выполненных в ходе обучения в МГТУ им. Баумана.

## 📁 Структура репозитория

Репозиторий организован по семестрам или отдельным дисциплинам. Каждая папка соответствует определенному курсу или практике.

*   **[Algorithms_and_Data_Structures](Algorithms_and_Data_Structures)** — решения некоторых задач из курса "Алгоритмы и структуры данных".
*   **[Assembly](Assembly)** — лабораторные работы по программированию на языке Ассемблера.
*   **[Computer_Networks](Computer_Networks)** — работы по курсу "Компьютерные сети".
*   **[Discrete_Math_and_Informatics](Discrete_Math_and_Informatics)** — некоторые задания по дискретной математике и основам информатики.
*   **[Graphics_Algorithms](Graphics_Algorithms)** — лабораторные работы по алгоритмам компьютерной графики.
*   **[Informatics_Basics_Scheme](Informatics_Basics_Scheme)** — основы информатики и программирования на языке Scheme.
*   **[Programming_Languages_and_Methods_Java](Programming_Languages_and_Methods_Java)** —  работы по курсу "Языки и методы программирования" на Java.
*   **[Summer_Practice](Summer_Practice)** — проект, выполненный в рамках летней практики после первого курса.

## ⚠️ Важное замечание о содержимом

Обратите внимание, что репозиторий находится в процессе наполнения. Некоторые папки могут содержать **неполный набор материалов** за соответствующий курс (например, только некоторые лабораторные работы или проекты). Это связано с естественным ходом учебного процесса и потерей некоторых работ.

## 🚀 Начало работы

Вы можете свободно клонировать репозиторий или просматривать отдельные папки, чтобы ознакомиться с примерами выполненных заданий.

```bash
git clone https://github.com/Mickleburg/BMSTU_Coursework.git
```
