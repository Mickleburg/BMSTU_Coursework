# P2P Memory (LAN)
Минимальная LAN‑сетка на WebSocket с хранением данных **в памяти (dict/list)** без CSV/файлов.
- Узлы обнаруживают друг друга в локальной сети (/24) и обмениваются таблицей узлов.
- Балансы и транзакции живут в оперативной памяти; при перезапуске — обнуляются.
- Нет устойчивости к недобросовестным узлам — это учебный пример.

## Быстрый старт
1) Установите зависимости:
```
pip install websockets==12.0
```
2) Запустите первый узел (создаст сеть):
```
python app.py --owner Lev --port 9000
```
3) На другом компьютере той же подсети запустите второй узел (сам найдёт сеть):
```
python app.py --owner Olga --port 9000
```
4) В меню выберите:
- 1 — показать таблицу узлов
- 2 — отправить коины

## Параметры
- `--owner` — имя владельца узла (обязательный параметр).
- `--port` — порт WebSocket сервера (по умолчанию 9000).
- `--subnet` — подсеть формата `192.168.1.`; по умолчанию извлекается автоматически из локального IP.

## Ограничения/планы
- Нет персистентности; при рестарте баланс/таблица сбрасываются.
- Нет криптографии и авторизации; доверяем сообщениям в пределах локалки.
- Транзакция защищена грубой блокировкой сети (lock), чтобы показать идею консенсуса.
